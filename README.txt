Weblocks Mustache Templates Editor

This is a piece of code which helps to manage mustache templates for your Weblocks project.
You can manage templates in your text editor and in web interface and synchronize your changes at any time.

You can use code to add content editing ability for site administrator.

Interface is added into Weblocks CMS admin part, so you need just to load package and it will work.
Other initialization instructions related to Weblocks CMS.

Package contains Weblcoks CMS model Template.
Usually only two buttons are added to every Weblocks CMS model gridedit - "Add" and "Delete".
Template gridedit has also "Import" and "Export" buttons for importing/exporting templates.
